to run the environment:

1.) install anaconda for python2.7
instructions on how to use anaconda: http://conda.pydata.org/docs/using/envs.html

2.) libraries to download:
    a.) install python-occ (0.18) (for all geometry operations)
    b.) conda install numpy (1.11.0) (BSD accepted license)
    c.) conda install scipy (0.17.0) (BSD accepted license)
    d.) pip install pycollada (BSD 3-clause "New" or "Revised" License)
	
3.) conda install spyder (2.3.8) (for editing the scripts)
